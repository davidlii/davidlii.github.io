Github访问慢？

Github为国外网站，所以域名解析时速度慢的。。解决此问题有俩方法：

  • 挂VPN
  • 修改hosts配置

本文讲的是第二种，修改hosts文件，具体步骤如下：

#1、打开host文件

  • 直接通过Finder

快捷键 command + shift + G，输入“/etc”，回车即可进入etc文件夹，找到hosts文件，直接通过系统的“文本编辑”打开即可。

  • 使用终端打开
