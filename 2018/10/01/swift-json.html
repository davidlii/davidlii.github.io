Swift中JSON转model

1.Codeble

Swift中，可以使用Codable协议来将JSON转换为model。

Codable 是一个组合协议，由 Decodable 和 Encodable 两个协议组成。作用是将 JSON 转换为 model 对象，也可以反过来将 model 对象转换为 JSON 或其它格式。

2.映射

如果Model中的属性名与 JSON 中的键不一致，可使用CodingKeys枚举来映射。
